JAVA类基础知识【JAVA学习笔记1】

Author: Yixuan Wan Date: September 28, 2023  9:03:10 Category: JAVA笔记

JAVA类结构和Main函数

JAVA类结构

  • JAVA 类名必须和文件名一致，大小写敏感
  • 一个JAVA文件可以有多个class（不提倡这么做），但只能有一个是public
  • 类是JAVA中最基础的逻辑单位
    • 所有内容必须放在类的范围中，不允许游离在类以外
    • 换言之，JAVA项目就是由一个个类组成的

Main函数

  • 一个class之多有一个main函数，也可以没有main函数
    • 没有main函数的类不能主动执行，但可以被动执行
  • 程序的入口都是main函数。和C/C++不同的是，JAVA要求所有的程序都必须放在类对象中
  • main函数的固定写法：PSVM（public static void main）
  • main函数不属于成员函数，也无法被其他方法/类所调用
  • main函数的形参：String[] args
    • 这些值是在main函数启动的时候输入的

基本类型和运算符

JAVA的8中基本类型

  • boolean
    • 取值只有true和false（大小写敏感），也不可以用数字赋值
  • byte
    • 主要用来代替小整数，1byte = 8 bits
    • 最小值-128 最大值127
    • 在二进制文件的读写中使用的比较多
  • short/int/long
    • short，16位（范围 -2^15 ~ 2^15-1）
    • int, 32位
    • long, 64位，long a = 100000000000L
    • 如果赋值越界，将会报错
  • float/double
    • float 32位
    • double 64位
    • float和double都不能用来表示很精确的数字 1.23==1.299999999
  • char
    • char是一个单一的16位的Unicode字符
    • 最小值 \u0000
    • 最大值 \uffff
    • 可以定义任何单一字符

JAVA运算符

  • 和C语言基本一致
  • 用异或运算实现值的交换
    • a = a ^ b;
      b = a ^ b;
      a = a ^ b;

选择和循环结构

选择结构

  • if
  • if…else
  • if…else if…else
  • switch
    • 多个case
    • 满足一个分支后，需要break
    • 最后一个分支为，default

循环结构

  • while
  • do..while
  • for

自定义函数

  • 函数必须放在类的范围内
  • 修饰词 返回值 函数名（形参列表）
    • 修饰词：public或者static
    • 返回值：int / void / …

重载Overload

  • 同一个类中，函数名称可以相同（即重载函数），但是函数参数的个数或者类型必须不同
  • 不能以返回值区分同名函数
